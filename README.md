# towerdefense2

Een eenvoudige start voor de `feature/enemy-movement`.

## Snel starten
- Start Laragon en open: http://localhost/towerdefense2
- Je ziet een canvas met een pad en een bewegende enemy.

## Belangrijke bestanden
- [index.php](index.php): HTML-skelet en canvas.
- [enemy_movement/style.css](enemy_movement/style.css): basisstijl.
- [enemy_movement/enemy.js](enemy_movement/enemy.js): eenvoudige movement-loop.

## Aanpassen
- Pas het `path` in [enemy_movement/enemy.js](enemy_movement/enemy.js) aan om de route te wijzigen.
- Verhoog of verlaag `enemy.speed` voor snelheid.
